# Screenshots cho Báo cáo

Folder này chứa các screenshots cần thiết cho báo cáo đồ án.

## Danh sách 8 Screenshots cần chụp:

| # | Tên file | Mô tả | URL/Nguồn |
|---|----------|-------|-----------|
| 1 | `login.png` | Form đăng nhập | http://localhost:8000/php/dangnhap.php |
| 2 | `dashboard.png` | Dashboard với 6 biểu đồ Chart.js | http://localhost:8000/php/dashboard.php |
| 3 | `booklist.png` | Danh sách sách với phân trang | http://localhost:8000/php/danhsachsach.php |
| 4 | `bookmanagement.png` | CRUD interface quản lý sách | http://localhost:8000/php/quanlysach.php |
| 5 | `cart.png` | Giỏ hàng mượn sách | http://localhost:8000/php/giohang.php |
| 6 | `docker_containers.png` | Docker Desktop - 3 containers | Docker Desktop App |
| 7 | `failover_terminal.png` | Test failover trong Terminal | Terminal chạy test-failover.sh |
| 8 | `mongodb_compass.png` | MongoDB Compass - Schema view | MongoDB Compass App |

## Hướng dẫn chụp:

1. Chạy script `screenshot_helper.sh` để mở các URLs tự động
2. Sử dụng phím tắt **Cmd + Shift + 4** (macOS) để chụp vùng chọn
3. Lưu ảnh vào folder này với tên file như trên

## Kích thước khuyến nghị:
- Web screenshots: 1920x1080 hoặc 1440x900
- Terminal: Đủ rộng để hiển thị output
- Docker Desktop: Full window
