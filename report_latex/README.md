## Lưu ý khi compile

1. **Vietnamese Support**: Sử dụng `babel-vietnamese` và `T5` font encoding

2. **Images**:
   - Đảm bảo symlink screenshots/ hoạt động
   - Hoặc copy trực tiếp từ ../screenshots/
   - Format hỗ trợ: PNG, JPG, PDF

3. **TikZ Diagrams**:
   - Có thể cần thời gian compile lâu hơn
   - Nếu lỗi, comment out các TikZ blocks

4. **Listings**:
   - PHP, JSON, YAML, JavaScript, Bash đã được config
   - Code tự động word-wrap

5. **Special Characters**:
   - Dollar sign trong MongoDB operators: dùng `\$match`, `\$group`...
   - Underscore trong tên biến: dùng `\_` hoặc trong \texttt{}

## Troubleshooting

### Lỗi "File not found: screenshots/..."
```bash
# Tạo symlink
cd /Users/tuannghiat/Downloads/Final\ CSDLTT/report_latex
ln -s ../screenshots screenshots
```

### Lỗi "Undefined control sequence"
- Kiểm tra package đã được include trong main.tex
- Kiểm tra TikZ libraries đã được load

### Lỗi Vietnamese characters
- Đảm bảo file được save với encoding UTF-8
- Kiểm tra `\usepackage[vietnamese]{babel}` và `\usepackage[T5]{fontenc}`
